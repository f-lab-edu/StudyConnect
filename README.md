### 📖 StudyConnect 프로젝트 소계 📖
배움을 좋아하는 사람들이 모여 함께 공부하는 스터디 플랫폼 <hola,클래스팅>

-----
### 🔍 주요 관심사 🔍
- 사용자의 편의성이 최대 목표
- 객체지향성을 극대화하여 확장에 용이한 개발 

-----
### 🌐 서버 구조 🌐
-----
### 🖥️ 사용 기술 및 환경 🖥️
- JAVA 17
- SpringBoot 3.0.5
- H2 DataBase
- MySql
